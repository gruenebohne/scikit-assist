uplift¶

skassist.evaluation.uplift.qini_curve(model, df, probabilities)¶

Calculates absolute and normalized Qini scores for conversion and revenue uplift. It also computes the conversion Qini relative to the optimal Qini score.

The following uplift curves are computed: #. Absolute conversion uplift curve #. Normalized conversion uplift curve #. Absolute revenue uplift curve
